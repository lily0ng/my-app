# Welcome

This Guides section is powered by markdown files.

## Add a new guide

1. Create a new file in `src/POST/guides/`, for example: `my-first-guide.md`
2. Paste the frontmatter template at the top
3. Write your content using markdown
4. Refresh the page — it will automatically appear in **Docs → Guides**

## Markdown features

- Tables (GFM)
- Task lists
- Code blocks
- Links

| Feature | Supported |
| --- | --- |
| Tables | Yes |
| Task Lists | Yes |
| Fenced code | Yes |
